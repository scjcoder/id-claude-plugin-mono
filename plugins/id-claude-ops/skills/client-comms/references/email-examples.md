# Email Examples — Sean's Voice

These examples demonstrate the preferred tone and structure for client-facing emails.
Reference these when calibrating tone or when Sean asks for refinement.

---

## Example 1: Issue Notification (First Communication)

**Context**: A sync issue was detected affecting a client's data feed.

```
Subject: Data Feed Sync — Action Being Taken

Hi [Name],

I wanted to give you a heads-up that we've identified a sync issue affecting your data feed. Our team is actively working on it and I'll keep you posted as things develop.

No action is needed on your end right now. I'll follow up as soon as I have more to share.

Sean
```

*Notes: No timeline given (first communication). No office name. Direct, not alarming.*

---

## Example 2: Follow-Up / Status Check

**Context**: Following up after a client hasn't responded to a previous message.

```
Subject: Following Up — [Topic]

Hi [Name],

Just circling back on my note from [day/date]. Wanted to make sure it didn't get buried.

[Restate the ask or key point in one sentence.]

Let me know if you have any questions.

Sean
```

*Notes: Non-accusatory, brief, restates the point clearly.*

---

## Example 3: Request for Information

**Context**: Needing the client to provide something before work can proceed.

```
Subject: Quick Question — [Topic]

Hi [Name],

To move forward on [topic], I need one thing from you:

- [Specific item needed]

Once I have that, I can get things moving on our end. Let me know if you have any questions.

Sean
```

*Notes: Single clear ask. No filler. Action-oriented subject.*

---

## Example 4: Good News / Completion Notice

```
Subject: [Project/Task] — Complete

Hi [Name],

Good news — [task/project] is done and ready on your end.

[One sentence on what was completed and what, if anything, they need to do next.]

Sean
```
