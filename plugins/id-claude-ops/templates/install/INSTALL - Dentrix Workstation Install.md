Our program can technically be installed and run from a workstation. However, installing it on a workstation instead of the office server could result in an unstable connection, as workstations are often shut down or restarted throughout the day.

For optimal reliability and stability, we strongly recommend performing the installation on the server. During the installation, we’ll need the Family File open to complete the setup properly.
