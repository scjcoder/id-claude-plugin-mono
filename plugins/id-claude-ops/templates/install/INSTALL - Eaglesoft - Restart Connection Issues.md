Then please follow in this order:

1. Right click on Dataco Connectivity Agent WATCHER and click STOP

2. Right Click on Dataco Connectivity Agent and Click Stop

3. Right click on Dataco Connectivity Agent and Click start/restart

       Wait Until Status says RUNNING

4. Right click on Dataco Connectivity Agent WATCHER and click start/restart
