# INSTALL - System Requirements

**HubSpot ID:** 3727853
**Shortcode:** #sysreq

## Content
InsideDesk Software Setup - IT Summary

• Installation Location: Install directly on the PMS server (Dentrix, Eaglesoft, or Dentrix Enterprise).

• Services: Ensure the DataCo Connectivity Agent and DataCo Connectivity Watcher services are installed and running.

• Port: Open port 443 for outbound communication (TLS 1.2 encryption).

• Data Transfer: Outbound-only, encrypted with TLS 1.2.

• Remote Access: No remote access required after installation.

Steps:

1. Install the software on the PMS server.

2. Verify the services are running and set to start automatically.

3. Ensure port 443 is open for outbound traffic.

4. Confirm data encryption is in place (TLS 1.2).

5. Disable remote access post-install if desired.
